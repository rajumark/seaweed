# Seaweed

A cross-platform (Windows, macOS, Linux) desktop application built with Dear ImGui. Seaweed is a super-powered ADB GUI tool designed for Android developers and device farm operators — wrapping every ADB command in a visual interface with well-organized data presentation.

## Goal

Provide a single, native desktop application that exposes the full power of ADB through an intuitive GUI, eliminating the need to memorize command-line flags while still offering complete control over connected Android devices and emulators.
